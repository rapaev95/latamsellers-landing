Perguntas
FAQ
É seguro enviar arquivos financeiros?▼
Sim. Todos os arquivos são processados localmente — não são enviados para nenhum servidor. Seus extratos ficam apenas no seu dispositivo.
Quais bancos e formatos são suportados?▼
Nubank PJ (CSV), C6 Bank BRL (CSV), C6 Bank USD (PDF, incluindo criptografados), Mercado Pago (CSV). Formato Mercado Livre: vendas_ml.xlsx.
Posso gerenciar múltiplos projetos?▼
Sim, sem limites. Cada projeto recebe DRE, Fluxo de Caixa e Balanço separados. Modos profit_share e rental para projetos com parceiros.
E se uma transação não for reconhecida automaticamente?▼
Transações não classificadas aparecem em uma lista separada. Você atribui manualmente a categoria e o projeto, e a regra é salva para os próximos uploads.
São necessários conhecimentos técnicos?▼
Não. A interface é feita para empreendedores sem conhecimento de TI: arraste o arquivo → receba o relatório.
Funciona com Shopee, JOOM, Estonia?▼
Sim. Suporta Shopee (SHPP), JOOM, projetos de serviços como Estonia (NFS-e, faturas). Cada um é mapeado para um projeto separado.
